# AMD Ryzen 安裝黑蘋果教學

本教學還在撰寫中，進度緩慢

最後更新日 - 2018 年 8 月 7 日

## 關於

由於安裝時遇到了非常多的阻礙，爬文爬遍了各地，包含了

* 中國黑蘋果集散地 - 遠景論壇
* 英語系 AMD 黑蘋果集散地
* Reddit Hackintosh版
* Tonymacx86
* 各部落格文章

就是因為爬了太多太多文章，所以在我裝好的那一刻我就有個念頭

> 我要寫一篇「從我的硬體出發，包含從系統安裝就驅動安裝的黑蘋果教學文章」

所以這個教學就這樣誕生了，希望能幫助到爬文爬到這邊的各位，教學中如果有任何問題，歡迎在 Github 上開 Issue 討論
